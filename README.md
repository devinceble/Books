## EJECT - Ejecting from Create React Native App
---

### I. Install react-native-cli and Eject
```bash
    # Using NPM
    $ npm install -g react-native-cli
    # Using YARN
    $ yarn global add react-native-cli

    # Eject
    $ yarn eject
```
